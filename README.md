## Install

1) Run in your terminal:

``` bash
git clone https://github.com/baldhadhaval08/laraveldemo.git laravel-demo
```

2) Set your database information in your .env file (use the .env.example as an example);

3) Run in your laravel-demo folder:
``` bash
composer install
php artisan key:generate
php artisan migrate
php artisan db:seed
```
